UDecide Decision Support System

NOTE - Application needs to be run in Google Chrome for file saving functionality

Instructions to run application...

If Google Chrome is your default browser:

   Double click the 'UDecide_Menu.html' file.


If Google Chrome is NOT your default browser:

   Right click 'UDecide_Menu.html' and select 'Open with > Google Chrome'
   OR
   Open Google Chrome and drag and drop 'UDecide_Menu.html' into the window
