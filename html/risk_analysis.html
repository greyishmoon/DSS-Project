Print to PDF

Save Project

The problem will be saved as <ProjectTitle>-DSS-Problem.json to your browsers download folder. After saving please manually move file to a permenant location for storage.

NOTE - Due to browser security previous saves of this project WILL NOT be overwritten automatically. Multiple saves of the same name will be appended (2)(3)(4) etc.

Load Project

Navigate to folder project saves were moved to and select <ProjectTitle>-DSS-Problem.json - All unsaved data from current project will be lost.
